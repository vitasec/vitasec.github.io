### Technical Profile

I am a security researcher and offensive security enthusiast specializing in vulnerability assessment and penetration testing. My focus lies in understanding the complexities of system architecture through an attacker's perspective to drive more robust security implementations.

### Core Competencies

*   **Offensive Security:** Web Application Pentesting, Network Exploitation, and Post-Exploitation techniques.
*   **Methodology:** Deep expertise in vulnerability research, including SQL injection, XSS, and LFI, with a focus on manual exploitation and automated tool-assisted workflows.
*   **Tooling & Frameworks:** Extensive experience with Burp Suite, SQLmap, ffuf, Nmap, and C2 frameworks such as Havoc, Mythic, and Sliver.
*   **Operating Systems:** Advanced proficiency in Linux environments, utilizing Arch Linux as a primary workstation, with specialized knowledge in Kali Linux and BlackArch for security testing.
*   **Continuous Improvement:** Active engagement in high-intensity security challenges on platforms like HackTheBox and TryHackMe, focusing on competitive "King of the Hill" scenarios and complex lab environments.

### Research Interests

My work focuses on documenting the lifecycle of security engagements—from initial enumeration and exploitation to post-exploitation and lateral movement. This space serves as a repository for technical write-ups, security research, and proof-of-concept explorations.
